Linux commands

  • pwd - present working directory
  • ls - list
  • mkdir - make directory/folder
  • cd 'Tab' - autofill suggested name
  • touch index.html - makes an empty file with the name index.html
  • rm index.html - delete file
  • rm -r 'filename' - delete all the files in the folder and delete the folder
  • mv 'old filename' 'new filename' - renames the file
  • cd .. - go back up in a directory

Git commands

  • git init - change folder to a GIT repository
  • git add ("." includes all files)
  • git commit -m - message describing commit
  • git checkout -b (-b makes new branch and switches to it)
  • git push

Docker commands

  • docker compose up - aggregates the output of each container
  • docker compose down - stops and removes containers, networks, volumes and images created by up
  • docker ps - lists the running containers
  • docker kill "container id" - shuts the container down
